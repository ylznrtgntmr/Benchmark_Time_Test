# Benchmark Time Test

## Project Description

This project measures and compares the execution time of different Python operations.
It is a simple performance testing project designed to analyze how long certain tasks take to execute.

## Requirements

* Python 3.10 or later

## How to Run

Run the following commands in the terminal:

python time_test1.py
python time_test_2.py


Each script will calculate and display the execution time of a specific Python operation.

## Tests Included

**zaman_test1.py**
Compares different methods of creating lists in Python.

**zaman_testi_2.py**
Measures the execution time of a loop operation.

Each script focuses on measuring how long a specific task takes to run in order to analyze Python performance.

## Purpose

The goal of this project is to understand and analyze Python performance by measuring execution times of different tasks.

## Example Output

Example output when running one of the scripts:

Execution Time: 0.0021 seconds

(The exact value may vary depending on the system.)
